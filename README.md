![screenshot](media/screenshot.png?raw=true "Screenshot")

Installation
------------

```
git clone https://github.com/Caleydo/ordino_public.git
cd ordino_public
npm install
```

Testing
-------

```
npm test
```

Building
--------

```
npm run build
```
